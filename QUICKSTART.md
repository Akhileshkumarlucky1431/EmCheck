# Quick Start Guide for AI Animation Video Creator

## Setup Instructions

1. **Prerequisites**  
   - Install Python 3.7 or higher.  
   - Install necessary libraries:  
     ```bash  
     pip install -r requirements.txt  
     ```  

2. **Clone the Repository**  
   ```bash  
   git clone https://github.com/Akhileshkumarlucky1431/EmCheck.git  
   cd EmCheck  
   ```  

3. **Run the Application**  
   ```bash  
   python app.py  
   ```  

## Example Prompts
- **Create a video on the topic of AI advancements:**  
   "Create a 2-minute animation explaining the recent advancements in AI technology."
- **Marketing Video Example:**  
   "Generate a promotional video for a new online course on digital marketing."

## Usage Guide

Once the application is running, you can input your prompt into the provided interface. The application will process your input and generate an animation video accordingly. You can customize the animation by choosing different styles and settings from the interface options.
