# Penetration Testing Tool for Email Security

A comprehensive penetration testing (pen testing) tool designed to identify vulnerabilities, misconfigurations, and security weaknesses in email systems. This tool provides automated security assessments, phishing simulations, and detailed reporting for email infrastructure.

## Features

- **SPF/DKIM/DMARC Analysis** - Validate email authentication mechanisms
- **Email Header Analysis** - Detect spoofing and header manipulation
- **Phishing Simulation** - Simulate phishing campaigns for security awareness
- **TLS/SSL Validation** - Check encryption configurations
- **Open Relay Detection** - Identify misconfigured mail servers
- **Comprehensive Reporting** - Generate HTML, PDF, and JSON reports
- **RESTful API** - Integrate with other security tools
- **CLI Tools** - Run scans from command line
- **Batch Processing** - Scan multiple domains efficiently
